3D Art

VTuber Model

A 3D .VRM VTuber model I made. Designed, modelled, textured and rigged all by myself.

Uses the MToon shader (default for VRoid models) and includes the 52 ARKit blendshapes, which is used in Apple's facetracking software.

Created with: Blender, paint.net, Krita and Unity.

Custom TF2 Class Portraits

I realized that the option to render live 3D class portraits was probably costing performance in TF2. So I used SFM to make new class portraits reflecting cosmetic items on my classes.

Made for both RED and BLU teams and supports Spy's cloak and disguises.

Created with: SFM, VTFEdit, and paint.net.
